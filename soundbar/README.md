# Soundbar Profiles

IR profiles for soundbars and sound bases.

## Structure

```
soundbar/
├── samsung/
│   └── samsung_hw_q_series.json
├── sony/
│   └── sony_ht_a_series.json
├── bose/
│   └── bose_soundbar_700.json
└── sonos/
    └── sonos_beam.json
```

## Common Commands

- `power`, `power_on`, `power_off`
- `volume_up`, `volume_down`, `mute`
- `input_hdmi`, `input_optical`, `input_bluetooth`
- `bass_up`, `bass_down`
- `treble_up`, `treble_down`
- `surround_on`, `surround_off`
