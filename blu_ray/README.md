# Blu-ray/DVD Player Profiles

IR profiles for Blu-ray players, DVD players, and disc changers.

## Structure

```
blu_ray/
├── sony/
│   └── sony_ubp_x_series.json
├── panasonic/
│   └── panasonic_dp_ub_series.json
├── lg/
│   └── lg_ubk_series.json
└── samsung/
    └── samsung_ubd_series.json
```

## Common Commands

- `power`, `power_on`, `power_off`
- `open_close` (disc tray)
- `play`, `pause`, `stop`
- `rewind`, `fast_forward`
- `previous`, `next` (chapter skip)
- `menu`, `popup_menu`, `top_menu`
- `up`, `down`, `left`, `right`, `select`
- `return`, `home`
- `subtitle`, `audio`
