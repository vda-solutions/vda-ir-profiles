# AV Receiver Profiles

IR profiles for audio/video receivers and amplifiers.

## Structure

```
av_receiver/
├── pioneer/
│   └── pioneer_vsx_series.json
├── denon/
│   └── denon_avr_x_series.json
├── yamaha/
│   └── yamaha_rx_v_series.json
└── onkyo/
    └── onkyo_tx_nr_series.json
```

## Common Commands

- `power`, `power_on`, `power_off`
- `volume_up`, `volume_down`, `mute`
- `input_tv`, `input_bd`, `input_cd`, `input_aux`, `input_phono`
- `input_hdmi1`, `input_hdmi2`, etc.
- `surround_mode`, `stereo`, `direct`
- `menu`, `up`, `down`, `left`, `right`, `select`
