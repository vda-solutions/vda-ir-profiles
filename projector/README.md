# Projector Profiles

IR profiles for home theater and business projectors.

## Structure

```
projector/
├── epson/
│   └── epson_home_cinema.json
├── benq/
│   └── benq_ht_series.json
├── optoma/
│   └── optoma_uhd_series.json
└── sony/
    └── sony_vpl_series.json
```

## Common Commands

- `power`, `power_on`, `power_off`
- `input_hdmi1`, `input_hdmi2`, `input_vga`
- `menu`, `up`, `down`, `left`, `right`, `select`
- `back`, `exit`
- `zoom_in`, `zoom_out`
- `focus_near`, `focus_far`
- `keystone_up`, `keystone_down`
- `blank`, `freeze`
- `aspect_ratio`
