# TV Profiles

IR profiles for televisions.

## Structure

```
tv/
├── samsung/
│   ├── samsung_qled_2023.json
│   └── samsung_frame_series.json
├── lg/
│   └── lg_oled_c_series.json
└── sony/
    └── sony_bravia_x90.json
```

## Common Commands

- `power`, `power_on`, `power_off`
- `volume_up`, `volume_down`, `mute`
- `channel_up`, `channel_down`
- `input`, `input_hdmi1`, `input_hdmi2`, etc.
- `menu`, `home`, `back`, `exit`
- `up`, `down`, `left`, `right`, `select`
- `0-9` (number keys)
