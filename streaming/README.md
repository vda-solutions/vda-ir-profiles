# Streaming Device Profiles

IR profiles for streaming media players and set-top boxes.

## Structure

```
streaming/
├── roku/
│   └── roku_ultra.json
├── amazon/
│   └── amazon_fire_tv.json
├── apple/
│   └── apple_tv_4k.json
└── nvidia/
    └── nvidia_shield.json
```

## Common Commands

- `power`, `power_on`, `power_off`
- `home`, `back`, `menu`
- `up`, `down`, `left`, `right`, `select`
- `play`, `pause`, `play_pause`
- `rewind`, `fast_forward`
- `volume_up`, `volume_down`, `mute`
