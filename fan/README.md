# Fan Profiles

IR profiles for ceiling fans, tower fans, and portable fans.

## Structure

```
fan/
├── hunter/
│   └── hunter_ceiling_fan.json
├── hampton_bay/
│   └── hampton_bay_ceiling_fan.json
├── dyson/
│   └── dyson_pure_cool.json
└── honeywell/
    └── honeywell_tower_fan.json
```

## Common Commands

- `power`, `power_on`, `power_off`
- `speed_low`, `speed_medium`, `speed_high`
- `speed_up`, `speed_down`
- `oscillate`, `oscillate_on`, `oscillate_off`
- `timer`, `timer_1h`, `timer_2h`, `timer_4h`, `timer_8h`
- `mode_normal`, `mode_natural`, `mode_sleep`
- `light`, `light_on`, `light_off` (for ceiling fans)
