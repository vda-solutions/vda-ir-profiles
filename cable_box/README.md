# Cable/Satellite Box Profiles

IR profiles for cable boxes, satellite receivers, and DVRs.

## Structure

```
cable_box/
├── directv/
│   └── directv_genie.json
├── xfinity/
│   └── xfinity_x1.json
├── dish/
│   └── dish_hopper.json
└── spectrum/
    └── spectrum_dvr.json
```

## Common Commands

- `power`, `power_on`, `power_off`
- `channel_up`, `channel_down`
- `volume_up`, `volume_down`, `mute`
- `guide`, `menu`, `info`, `exit`
- `up`, `down`, `left`, `right`, `select`
- `0-9` (number keys)
- `record`, `play`, `pause`, `stop`
- `rewind`, `fast_forward`
- `dvr`, `on_demand`
